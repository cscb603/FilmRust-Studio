# 我把 Kodak、Fujifilm、Ilford 的胶卷，做成了一个软件

你有没有这种感觉？

手机拍了照片，总觉得少了点什么。

少了什么呢？我自己想了很久——是「质感」。数码照片太"干净"了，干净得不像真的。颜色太准确了？不，是**太标准了**。

而胶片不一样。

胶片的颜色会"偏"——正是这种偏，才是好看的。

Kodak Portra 的肤色，是微微发暖的，像记忆里下午的阳光。Fujifilm 的绿色，带着一点青，像日本的夏天。Ilford 的黑白，不是单纯去色，而是有银盐颗粒在呼吸。

这些味道，我做成了软件。


## 它不是滤镜

市面上绝大多数"胶片滤镜"，其实就是调了一组曲线，改了一下饱和度。

我们不一样。

这个软件叫 **FilmRust Studio**，它的底层是一个物理引擎——是真的在模拟胶片的三层感光乳剂怎么感光、怎么化学反应、怎么显影。

所以它做出来的效果，不是"看起来像"，而是**算出来就是**。

比如 Kodak Portra 400：我们用它的真实光谱数据、它的 D-max 密度、它的特性曲线来算颜色，不是调色板里拖两个滑块。

这有什么区别？

你拿一张普通数码照片，用普通的滤镜套一下——嗯，好像有点胶片味了。

再用 FilmRust 跑一下——**完全不一样**。高光的颜色是有厚度的，阴影不是一块死黑，而是有层次的深色，颗粒不是数字噪点，是真的像显微镜底下看到的银盐晶体。

不懂这些没关系。你试一下就知道了。

## 怎么用？三种姿势

### 第一种：丢进去，点一下

电脑上双击打开，把照片拖进去，左边选一个胶片，点「开始显影」。

然后等一两秒，就能看到"冲洗"出来的效果。

效果出来后还能继续调：在色彩层调温暖／冷色，曲线层拉出各种曲线，觉得肤色不对劲还有专门的肤色优化。

深色/浅色界面会按时间自动切换——晚上打开就是深色，不刺眼。点一下按钮也能手动锁定。

### 第二种：在 Photoshop 里面用

装好之后，PS 菜单 → 文件 → 脚本 → 胶片调色，就弹出一个面板。

它会先分析你的照片，自动给你推荐最合适的胶片。

比如你拍的是人像，它就会推荐 Portra。拍的是夜景，它就知道推荐 CineStill 800T。

选好之后点确定，胶片效果就贴到你的图层面板里了，不弹窗不打扰。

如果平时处理很多，可以设个快捷键 Ctrl+Shift+F，以后一键出片。

### 第三种：写代码调用

这个主要是给技术型玩家用的。命令行一句话，就能批量处理几百张照片。

有些朋友把它接入了 ComfyUI 或者自己写的自动化脚本里，每天自动处理。


## 最新 v6.3 更新了什么

这个版本做了几件大事：

**颗粒控制全面简化了。** 以前我们有颗粒的控制分支逻辑——什么时候降噪、什么时候加强——结果在某张图上反而把画质搞坏了。直接全删了。现在胶片出什么样的颗粒，就是胶片物理数据决定的，我们不额外加东西。**原始胶片该长什么样，就长什么样。**

**图片导出质量拉到 97。** 之前有朋友导出 JPG，发现原图 10MB 处理完只剩 2MB——检查发现是导出 JPEG 默认质量只有 75，对细节密集的胶片照片压缩太狠了。现在改成摄影级质量 97，保留了胶片颗粒和色彩渐变的全部细节。

**支持 16 位 TIFF。** 很多专业摄影师用高位深 TIFF 拍摄，以前读不了，现在直接拖进去就能处理。

**界面明暗主题会看时间了。** 晚上 6 点到早上 6 点，自动切深色模式，白天自动浅色。当然如果你喜欢手动，点一下按钮就能锁定。

**深扒了几个人人都会踩的坑：**
- 左面板用滚轮切胶片预设不再干扰中间图片的缩放
- 显影动画结束后，不会再偷偷多合成一次导致画质下降
- 移除了一批画质不稳定的预设


## 一些小事

这个软件的主干，是 Rust 写的。Rust 是一门系统编程语言——性能好、内存安全。我们选择它，是因为胶片模拟涉及到大量的像素计算，丢给 Python 跑太慢，丢给纯 C 写又太容易出 bug。Rust 刚好。

它也不是一个人的东西。我们用了一个开源的胶片模拟引擎 filmr，开发者是国外一位叫 Carlos 的摄影师兼程序员。我们的工作是在它上面搭了图形界面、Photoshop 脚本、图层系统、智能推荐……

对，就是这个意思。
